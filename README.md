# Assignment S106

## General info:

**Project description**:
Creates generic methods that accept arguments of any kind.
Combines generic objects with specific types, including limitation with interfaces or classes.
Uses generic varargs and tests their implications.

**Technologies**: Backend Java

**Installation and Execution**:
1. Clone repository:
   https://github.com/bmestres/tascaS106.git
